# Volleyball Stats Recorder

バレーボールの試合スタッツを iPad / スマホで記録する Web アプリです。
インストール不要、登録不要、完全オフライン動作。

🌐 **アプリを開く**: https://slothfront.github.io/volleyball-stats/

---

## なぜ作ったか

高校教師として高校女子バレー部を指導するなかで、試合のデータを記録・分析できるツールが欲しくなりました。
既存の高機能ソフトは PC 前提・有料・操作が複雑で、部活レベルでは使いづらい。さらに調べてみると、機能が充実しているものはほとんどが英語など海外製のアプリで、日本語に完全対応した使いやすいツールがほぼ存在しないことに気づきました。

そこで、マネージャーが iPad でタップだけで日本語のまま記録できる 試合記録ツールを自分で作ることにしました。を自分で作ることにしました。

## できること

### 記録
- 自エース / 自スパイク決定 / 自ブロック決定 / 相手ミス
- 被サービスエース / 被スパイク失点（ゾーン記録）
- 自サーブミス / 自スパイクミス / その他自ミス / 自反則
- スパイク試行（ラリー継続中）
- 9 ゾーンへの落下位置を記録
- ローテーション自動管理（5-1 システム想定）
- リベロ対応
- 詳細記録モード（ミス内訳・反則種別を選択可）

### 分析
- セット / 試合のサマリー（得点内訳・SO% / PS%）
- ローテーションごとのサイドアウト率・ブレイク率
- 9 ゾーンヒートマップ（タイプ別 × ローテ別フィルタ）
- 個人スタッツ（サーブ効果率・スパイク効果率含む）
- ミス内訳（詳細モード時）

### データ管理
- 試合データは端末の localStorage に保存（オフライン動作）
- CSV エクスポート（ラリーログ / サマリー）
- JSON バックアップ・インポート
- スタメンプリセット保存
- ロスター（チームメンバー）の永続保存

## 使い方

### 初回

1. 上記 URL を iPad / スマホ / PC のブラウザで開く
2. iPad / iPhone なら Safari の「ホーム画面に追加」でアプリのように使える
3. 一度開けばオフラインでも動作する

### 試合記録の流れ

1. **新しい試合をはじめる**
2. 対戦相手と試合日を入力
3. 出場メンバーを入力（次回以降は「保存」で自動ロード）
4. スタメン 6 人を P1〜P6 に配置
5. 最初のサーブ権を選択
6. 試合中、得点が動くたびに該当ボタンをタップ
7. セット終了 / 試合終了の操作で集計画面へ

### データの保存

- 端末の localStorage に保存されるため、**ブラウザの履歴 / キャッシュをクリアすると消えます**
- 試合ごとに JSON バックアップを取ることを推奨
